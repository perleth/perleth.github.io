Apperle is a Clean and Modern Responsive App Landing page template. It is made using materialize which gives it a unique custom design . It works nicely on all smart devices: smart phones, tablet, PCs and desktops. It includes 28 Color options to help you choose the right color for your app and reusable components, so that you can easily create new layouts with these components. Template includes built in Sass support , which help to change the color easily. All files are clearly organised we believe it will be easy to use and edit them. Any one can customize this template as his wishes.
